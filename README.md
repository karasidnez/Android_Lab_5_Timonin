# Лабораторная работа №5: Jetpack Compose - контейнеры, ARGB-модель и Material Design

## Цель работы
Повторить основы разработки пользовательского интерфейса на Jetpack Compose, изучить работу с контейнерами `Row` и `Column`, познакомиться с ARGB-моделью цветов, компонентом `Scaffold` и принципами Material Design.

## Краткое описание приложения
В ходе выполнения лабораторной работы было создано простое приложение, демонстрирующее ключевые возможности Jetpack Compose по управлению компоновкой элементов на экране.

**Функциональность и особенности:**
*   **Демонстрация контейнеров:** Приложение использует `Column` в качестве основного контейнера для вертикального расположения элементов интерфейса.
*   **Горизонтальное размещение:** Внутри главного контейнера используется `Row` для создания горизонтального ряда с тремя цветными квадратами, что наглядно показывает разницу между вертикальной и горизонтальной компоновкой.
*   **Работа с ARGB-цветами:** Цвета для элементов заданы в ARGB-формате. Один из квадратов выполнен полупрозрачным для демонстрации работы альфа-канала (прозрачности).
*   **Информационный блок:** Присутствует текстовый заголовок и составной контейнер, который содержит вложенные друг в друга `Row` и `Column`, иллюстрируя создание более сложных и гибких макетов.
*   **Структурированный код:** Пользовательский интерфейс разбит на отдельные, переиспользуемые `@Composable`-функции, что соответствует современным практикам разработки под Android.
